Dashboard Overview

Selamat Datang di Dashboard

Ini adalah contoh halaman dashboard dengan sidebar yang responsif. Sidebar dapat diperkecil dan diperbesar sesuai kebutuhan.

Cara Menggunakan

1. Klik tombol di sidebar untuk mengecilkan/memperbesar sidebar

2. Klik tombol di header untuk membuka/menutup sidebar pada perangkat mobile

3. Klik ikon bulan/matahari untuk mengganti tema gelap/terang

Fitur Sidebar

✓ Responsif untuk desktop dan mobile

✓ Mode terang dan gelap

✓ Submenu yang dapat diperluas

✓ Status aktif untuk item menu

✓ Animasi halus
